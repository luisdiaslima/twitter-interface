## :bookmark: Sobre

A <strong>Interface do Twitter</strong> é uma aplicação web feita em Typescript e ReactJs.

Essa aplicação foi promovida e distribuída pela [Rocketseat](https://rocketseat.com.br/). A ideia de clonar a interface do discord veio da série <strong>UI Clone</strong>

## :fire: Como usar

- ### **Pré-requisitos**

  - É **necessário** possuir o **[Node.js](https://nodejs.org/en/)** instalado na máquina
  - Também, é **preciso** ter um gerenciador de pacotes seja o **[NPM](https://www.npmjs.com/)** ou **[Yarn](https://yarnpkg.com/)**.

1. Faça um clone :

```sh
  $ git clone https://github.com/thisluis/twitter-interface.git
```

2. Executando a Aplicação:

```sh
  # Instale as dependências
  $ npm install

  # Inicie a aplicação web
  $ npm start
```
